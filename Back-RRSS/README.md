# Back-RRSS
Back-RRSS is a simple script to backup your RSS feeds. It uses the [Feedparser](
    npm install,
    npm start
    ) library to parse the feeds and [Node.js](https://nodejs.org/en/) to run the script.

## Installation
    
        npm install

## Usage
    
            npm start
